# RepCo
Projet de représentation des connaissances, Licence3-Informatique

Algorithmes : 	- Largeur
				- Profondeur
				- A*

Ordre des actions de déclaration : D G H B

Fonctionnalités : 	- La largeur et la hauteur qui sont demandés correspondent à la grille du labyrinthe.
					- La grille est initialisée vide (toutes les cartes sont des passages).
					- Il faut obligatoirement un départ et une arrivée avant pouvoir lancer un algo.
					- Il est possible d'ajouter des murs, de voir le temps d'éxecution de l'algo et le nombre d'état 				qui 					correspond au nombre de carte visitée.
					- En bleu foncé est représenté le chemin de D (depart) à A (arrivée).
					- En bleu clair est représenté l'historique (cartes visitées)
